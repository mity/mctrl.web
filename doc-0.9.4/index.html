mCtrl Documentation

mCtrl is C open source library for MS Windows, providing a set of user interface controls, that complement the standard set as offered by USER32.DLL and COMCTL32.DLL libraries.

Please check project homepage http://mctrl.sourceforge.net and https://sourceforge.net/projects/mctrl for newer versions and more actual information.
